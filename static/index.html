3. Available Tools & Usage Context

These tools are exposed to the AI Agent via MCP. The Agent calls them based on the user's conversation.

check_payment_status

üîç When to use: The Agent calls this when a user asks "Where is my payment?" or "Check my history".

Because this is a READ operation, the Gateway auto-approves it (Low Risk).

{
  "name": "check_payment_status",
  "arguments": { "beneficiary_id": "string" }
}

request_payment_extension

‚ö†Ô∏è When to use: The Agent calls this if a user says "I can't pay on time" or "I need more time".

{
  "name": "request_payment_extension",
  "arguments": { 
    "beneficiary_id": "string", 
    "reason": "string" 
  }
}
