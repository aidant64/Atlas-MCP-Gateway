Integration Guide & MCP Schema

How to connect an AI Agent to the ATLAS Governance Gateway.

1. Connection Details

Configure your MCP Client (e.g., Claude Desktop, Cursor, or Custom Agent) with these settings:

  • Type: SSE (Server-Sent Events)
  • URL: https://atlas-mcp-gateway-vercel.vercel.app/mcp/sse
  • Headers: {"Authorization": "Bearer <YOUR_API_KEY>"}

3. Available Tools (Schema)

check_payment_status

Retrieves payment history. (Low Risk - Auto Approved)

{
  "name": "check_payment_status",
  "arguments": {
    "beneficiary_id": "string"
  }
}

request_payment_extension

Request a deadline extension. (High Risk - Triggers Governance)

{
  "name": "request_payment_extension",
  "arguments": {
    "beneficiary_id": "string",
    "reason": "string"
  }
}

modify_welfare_record

Update beneficiary details. (Critical Risk - Requires Approval)

{
  "name": "modify_welfare_record",
  "arguments": {
    "beneficiary_id": "string",
    "changes": "object"
  }
}
